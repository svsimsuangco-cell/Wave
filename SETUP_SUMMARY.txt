## 🚀 3-STEP QUICK START

### Step 1: Update MongoDB Connection
Edit `.env.local` and add MongoDB URI:

**Option A: MongoDB Atlas (Cloud - Recommended)**
```env
MONGODB_URI=mongodb+srv://username:password@cluster.mongodb.net/hosting-platform?retryWrites=true&w=majority
```
→ Get string from: https://www.mongodb.com/cloud/atlas
→ Full guide: See `MONGODB_ATLAS_SETUP.md`

**Option B: Local MongoDB**
```env
MONGODB_URI=mongodb://localhost:27017/hosting-platform
```
→ Install from: https://www.mongodb.com/try/download/community

**Pick ONE method:**

## ✨ How It Works

### User Registration Flow
1. User registers at `/auth/register`
2. Email & password hashed
3. Verification token generated (24-hour expiry)
4. User redirected to `/verification-pending`
5. Email verification link sent
6. User clicks link → account verified
7. Can now login at `/auth/login`
8. After login → `/dashboard` (user area)

### Admin Login Flow
1. Admin visits `/admin/login`
2. Enters email & password
3. JWT token generated
4. Sets `admin_auth` cookie
5. After login → `/admin/dashboard`
6. Can manage orders, services, plans
